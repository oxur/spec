# spec

*The Oxur Rust Lisp specification*

[![][logo]][logo-large]

## Dependencies

- mdbook (use `cargo install mdbook` to install it).

## Build steps

Generate the book:

```bash
mdbook build
```

The generated HTML will be in the `book` folder.
